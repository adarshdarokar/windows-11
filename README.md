# Windows 11 Portfolio

A modern and interactive Windows 11-inspired portfolio built with React, TypeScript, Tailwind CSS, and Zustand. The project recreates the Windows desktop experience inside the browser while showcasing my professional profile, skills, education, experience, projects, and contact information.

## Live Demo

🔗 Live: https://windows-11-eghg.onrender.com

## Features

* Windows 11 inspired user interface
* Fully responsive design
* Interactive desktop icons and folders
* Working taskbar with application management
* Resume viewer with PDF download support
* About, Contact, Education, Experience, Skills, and Projects sections
* Multiple wallpapers with live switching
* Accent color customization
* Light and Dark theme support
* System settings panel
* Sound controls
* Persistent state management using Zustand
* Smooth animations and modern UI experience
* Fully functional desktop environment simulation

## Tech Stack

* React
* TypeScript
* Tailwind CSS

## Installation

```bash
git clone <repository-url>
cd windows-11-portfolio
npm install
npm run dev
```

## Build

```bash
npm run build
```
